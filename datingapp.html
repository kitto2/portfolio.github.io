Dating -app Case Study

This dating app is designed to connect individuals based on shared interests, preferences, and astrological compatibility. The app allows users to create personalized profiles, input Kundali details for astrology-based matchmaking, and choose their interests to enhance match recommendations. The app offers a user-friendly interface with features like swipe-based matching, customizable filters for preferences, and a seamless chat system for interactions. It combines modern matchmaking techniques with a personalized touch, ensuring a tailored and engaging experience for users.

The Key Problem

  • Users drop off during onboarding due to lengthy or complex steps.
  • Some users prefer astrology-based matching, while others focus on personal interests.
  • Fake profiles and lack of verification reduce user trust
  • Kundali matching can be confusing for users unfamiliar with astrology.

The Solution

  • Implement a step-by-step onboarding process with optional Kundali input.
  • Allow users to choose between astrology-based, interest-based, or hybrid matchmaking.
  • Introduce ID verification, social media linking, and AI-driven profile moderation./li>
  • Provide clear explanations, tooltips, and an option for astrologer consultations./li>

key pages



Onboarding page

The onboarding screens provide a smooth and engaging start. The welcome screen features a warm design, tagline, and options for Sign Up or Log In, including social login for quick access.
The sign-up and login screens ensure easy authentication. The sign-up screen collects user details, while the login screen allows returning users to log in with credentials or social accounts. A "Remember Me" option enhances convenience, making the process seamless and user-friendly.

Profile Setup page

The profile setup process is divided into three simple steps for a smooth onboarding experience.
Basic Information: Users enter their full name, address, date of birth, and select their gender. A profile picture can also be uploaded.
Kundali Details: Users can input their birth time, birth address, and birth date for astrology-based matchmaking. This step includes a Skip option for flexibility.
Interests Selection: Users choose from various interests like music, travel, gaming, and fitness, helping to personalize their experience and improve match recommendations.

Homepage page

The homepage offers a smooth matchmaking experience with an inviting start screen and a "Start Exploring Profiles" button. Users can easily set their preferences through filters, adjusting Gunas compatibility, age range, and gender interest.
The match discovery screen presents potential matches with their profile details, including age, interests, and compatibility score. Users can interact by liking or skipping profiles, making the experience engaging and personalized.

Browsing screens

This screen helps users discover potential matches. Each profile shows basic details like age, job , location , and hobbies . Short phrases like "Design enthusiast" give a quick sense of personality. The design uses simple cards and tags to make profiles easy to skim. Overall, it’s built for fast browsing with clear, bold headings and minimal clutter.

Favourites page

This screen organizes user connections into three tabs: Previous Matches History, New Matches, and Recommended. The Previous Matches History lists past interactions, showing profiles and snippets of old chats for easy reference. The New Matches tab highlights recent connections, marked with notifications to grab attention. The Recommended tab suggests profiles based on shared interests or location.

chat page

This is where users chat with matches. On the left, a list shows recent conversations with unread message counts. The main chat window displays messages and offers fun prompts like "Truth" or "Dare" to start conversations. A keyboard and quick-reply buttons make typing easy. The design is clean, with timestamps and a focus on keeping chats lively and simple to navigate.

profile page

This screen lets users manage their personal information. They can edit their name, address, birthday, and bio . Special sections like "Kundali Details" ask for birth time and address, likely for astrology-based features. Settings like changing passwords or themes are neatly grouped under categories like Account and General. The layout is organized, with clear labels and forms, making it easy to update details securely.
